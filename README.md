# 📊 Streaming Platform EDA (Amazon Prime Dataset)

## 🔎 Project Overview
This project performs Exploratory Data Analysis (EDA) on a streaming platform dataset to uncover content trends, rating distribution, genre patterns, and country-wise production insights.

The objective is to clean the dataset, handle missing values effectively, and generate meaningful business insights using visualizations.

---

## 🎯 Objectives
- Data Cleaning & Preprocessing
- Handling Missing Values Strategically
- Movies vs TV Shows Comparison
- Genre Distribution Analysis
- IMDB & TMDB Rating Analysis
- Country-wise Content Distribution
- Correlation Analysis

---

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## 📈 Key Insights
- Movies dominate the platform compared to TV Shows.
- Most content has an IMDB rating between 6 and 8.
- Drama and Comedy are among the most frequent genres.
- Content production has significantly increased after 2010.
- Certain countries contribute more content than others.

---

## 🧹 Data Cleaning Approach
- Removed unnecessary columns
- Treated missing categorical values appropriately
- Ensured no null values remain in the final dataset
- Verified data types before analysis
