# Streaming Platform EDA

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a streaming platform dataset (Amazon Prime dataset).

## Objectives
- Data Cleaning
- Handling Missing Values
- Genre Analysis
- Rating Analysis (IMDB & TMDB)
- Country-wise Content Distribution

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Movies dominate over TV shows in total content.
- Certain genres are more frequent.
- Ratings distribution shows majority content between 6–8 IMDB score.
